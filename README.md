# 🚀 Career Guidance AI

An intelligent **AI-powered Career Guidance System** that helps users discover suitable career paths based on their **skills, interests, and educational background**.  
It acts as a virtual career counselor, providing personalized recommendations using AI/ML techniques.

## 📖 Overview

Choosing the right career path can be challenging. This project uses **Artificial Intelligence and Machine Learning** to analyze user inputs such as skills, interests, and education, and then recommends the most suitable career options.

## ✨ Features

- 🔍 Personalized career recommendations  
- 🧠 Skill and interest-based analysis  
- 💬 Simple and interactive UI  
- 📊 AI/ML-based prediction system  
- 🛣️ Career path suggestions

## ⚙️ How It Works

1. User enters:
   - Skills  
   - Interests  
   - Educational background  

2. Backend processes the data:
   - Cleans and formats input  
   - Passes data to ML model  

3. ML Model:
   - Matches user profile with trained dataset  
   - Predicts best career options  

4. Output:
   - Displays recommended career paths  
   - Provides guidance

## 🚀 Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Shreyas0951r/CarrerGuidanceAI.git
cd CarrerGuidanceAI
```
### 2️⃣ Create virtual environment
```bash
python -m venv venv
venv\Scripts\activate   # For Windows
```
### 3️⃣ Install dependencies
```bash
pip install -r requirements.txt
```
### 4️⃣ Run the application
```bash
python app.py
```
### 5️⃣ Open in browser
```bash
http://127.0.0.1:5000/
```
